It walks! It talks!

David Rutland November 13, 2024

Yet another switcharoo, and I've settled on AGS for the game engine. There are several reasons for this:

  1. The documentation is exhaustive, with many people compiling their own manuals, as well as video tutorials of varying quality.
  2. It comes with some sensible demo modes and prebuilt mockups, allowing me to easily choose an interface type.
  3. There's already an inventory system as well as save/load functionality.
  4. Anything else would probably be overkill.

It's pretty easy to get the character to walk behind objects, make areas clickable, and it just seems... straightforward.
